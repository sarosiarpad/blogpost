## Projekt klónozása
``` git clone https://github.com/your-username/blogpost.git ```

## Szerver oldali setup
**`.env` fájl** létrehozása, a szükséges paraméterekkel:
```env
   DB_USER=root
   DB_PASSWORD=your_password
   PORT=5000
   ```

Backend indítása
```
  cd server
  npm i
  node index.js
  ```

---

## Kliens oldali setup
```
  cd client
  npm i
  npm run dev
  ```
